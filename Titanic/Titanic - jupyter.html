Neste trabalho tentaremos responder as seguintes questes:

  • Quantas mulheres e homens estavam a bordo do navio.
  • Qual foi a mortalidade em cada sexo.
  • Qual era a faixa etria da tripulao
  • Qual foi a mortalidade por classe.
  • Qual o total de passageiros por local de embarque.
  • Qual foi a mortandade por local de embarque

def faixa_etaria(linha):
    '''
    Verifica o valor na coluna 'Age' e retorna uma string de acordo com a faixa de idade.

    Parameters:
    linha -- cada linha do data frame.

    Returns:
    string de acordo com a faixa etria: 'child' se < 18 // 'elderly' se >= 65 // 'adult' para qualquer outro valor
    
    '''     

    if linha.Age < 18:
        return 'child'
    elif linha.Age >= 65:
        return 'elderly'
    else: 
        return 'adult'

df['Faixa_Idade'] = df.apply(faixa_etaria, axis=1)

Conclusões:¶

  • Haviam mais homens do que mulheres no navio (577 homens e 314 mulheres). Entre os dois grupos, os homens tiveram, ainda que mantidas as devidas propores, uma mortandade muito maior. Provavelmente devido politica "mulheres e crianas primeiro".
  • Observamos que o ndice de mortalidade na terceira classe o mais alto. E que mesmo sendo o grupo predominante no navio, com larga vantagem para os outros dois, mesmo assim seu ndice de sobrevivncia foi baixo se comparado s outras classes. Observamos tambm que a primeira classe teve mais sobreviventes do que mortos. Alguns fatores podem ter contribuido para isso. Desde arbitrariedade por parte da tripulaco ao selecionar passageiros para entrarem nos botes, ou proximidade das cabines de primeira classe com os botes salva vidas, ou ambos.
  • A grande maioria dos tripulantes era oriunda de Southampton, que teve tambm a maior mortandade. Percebemos tambm que o grfico de sobrevivncia por local de embarque bastante semelhante ao grfico de sobrevivncia por classe.

Limitações:¶

Um fator a ser considerado de que os dados aqui apresentados no representam a realidade exata dos passageiros do Titanic, uma vez que existiam aproximandamente 1300 passageiros alm da tripulao. Aqui trabalhamos com dados de 891 pessoas. O trabalho apresentado no grfico 'Faixa Etria' claramente o mais impreciso, uma vez que muitos valores faltantes foram transformados na mdia das idades conforme introduzido na etapa de data wrangling. A coluna 'Cabin' apesar de ter muitos valores faltantes, por no ter sido utilizada na anlise, optou-se por no alter-la. Dois valores ausentes na coluna 'Embarked' foram preenchidos com 'Southampton' por ser o valor mais provvel. Passageiros oriundos do local representam mais do que o dobro do segundo lugar.
